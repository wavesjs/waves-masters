# waves-masters

> Low level components for transport and scheduling

## Documentation

[http://wavesjs.github.io/waves-masters/](http://wavesjs.github.io/waves-masters/)

## Usage

#### Install

```sh
npm install [--save] waves-masters
```

#### import

```js
import * as masters from 'waves-masters';
```

## List of components

#### core

- `PriorityQueue`
- `SchedulingQueue`
- `TimeEngine`

#### masters

- `PlayControl`
- `SimpleScheduler`
- `Scheduler`
- `Transport`

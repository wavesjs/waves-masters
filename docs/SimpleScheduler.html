Class: SimpleScheduler

SimpleScheduler

The SimpleScheduler class implements a simplified master for time engines (see TimeEngine) that implement the scheduled interface such as the Metronome and the GranularEngine. The API and funtionalities of the SimpleScheduler class are identical to the Scheduler class. But, other than the Scheduler, the SimpleScheduler class does not guarantee the order of events (i.e. calls to the advanceTime method of scheduled time engines and to scheduled callback functions) within a scheduling period (see period attribute).
